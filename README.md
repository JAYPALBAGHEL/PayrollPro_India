# Indian Payroll App

A lightweight payroll calculator for Indian companies.

## Features
- Calculates salary breakup (Basic, HRA, Special Allowance)
- Employee PF and employer PF estimates
- Professional tax and monthly TDS estimate
- Supports old and new tax regimes (simplified slab logic)
- Shows estimated monthly in-hand salary and employer monthly cost

## Run locally
```bash
python3 -m http.server 8000
```
Then open `http://localhost:8000`.

## Notes
- This tool provides estimates only and should not be treated as legal/tax advice.
- Tax rules can change by financial year and state-specific regulations.
